# Alien Invasion Custom

A 2D space shooter game built with Python and Pygame. This project is a customized version of the "Alien Invasion" game from the *Python Crash Course* book.

## Features
- Player-controlled ship with smooth movement.
- Alien fleet that descends and shifts.
- Scoring system and level progression.
- Developed on **Linux** using **gnome-text-editor**.

## Prerequisites
- Python 3.x
- Pygame library

## Installation & Running
1. Clone the repository:
   `git clone https://github.com/bayazit/alien_invasion_custom.git`
2. Create and activate a virtual environment:
   `python3 -m venv venv`
   `source venv/bin/activate`
3. Install dependencies:
   `pip install -r requirements.txt`
4. Run the game:
   `python3 alien_invasion.py`
